Bienvenido a nuestra tienda virtual

En el presente trabajo hace parte de la asignatura Desarrollo de Software II y en este se detallará el proceso de instalación de los programas necesarios para el desarrollo de una página web para la empresa ficticia Medellín Outfiters, esta empresa se dedica a la comercialización de ropa.

Para el desarrollo de dicha página se utilizarán herramientas fundamentales en el mundo del desarrollo web como Visual Studio Code, Node JS y Git/GitHub.

El objetivo principal de la página web será contar con un sistema de administración que permita realizar las operaciones básicas de un CRUD
